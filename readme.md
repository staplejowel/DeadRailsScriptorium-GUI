## üì¶ Installation Guide

Getting started is easy! Follow these steps to install and launch the Dead Rails Script GUI on your system.

**1. Download Loader.rar from the repository.**  
Navigate to the releases tab in this Github repository and download the latest `Loader.rar` file.

**2. Extract the Loader.rar file.**  
Use a reliable archive extraction tool such as WinRAR, 7-Zip, or Keka to extract the contents of the archive to your preferred folder.

**3. Launch the Dead Rails Script GUI executable.**  
Double-click the application executable that matches your OS:
- For Windows: `DeadRailsScriptGUI.exe`
- For Linux: `DeadRailsScriptGUI.AppImage` or `.deb` package
- For macOS: `DeadRailsScriptGUI.app`

**4. (Optional) Install Required Dependencies.**  
The program will prompt you if any common libraries or run-times are missing. Most major systems will have everything needed pre-installed.

**5. Run & Explore!**  
Start creating, loading, and managing Dead Rails scripts through the friendly GUI.
